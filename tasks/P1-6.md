# P1-6 — Add menu (Shift-A), Duplicate (Shift-D), Delete (X)

Read `tasks/P1-CONVENTIONS.md` first. Depends on P1-1 (`PRIMITIVES` registry)
and P1-2 (keymap patterns in InputManager) — both are already merged.

## Files
- `src/ui/addMenu.ts` — new: DOM popup menu
- `src/input/InputManager.ts` — edit ONLY `onKeyDown` (and a private field for the open menu)
- `src/main.ts` — only if strictly necessary (prefer not to touch it)

## Add menu (Shift-A)
- `Shift+A` (no ctrl/alt, no active operator) opens a small popup at the current
  pointer position inside `#viewport-wrap`, listing every entry from
  `PRIMITIVES` (`src/core/mesh/primitives.ts`) under a "Add Mesh" heading.
- Class the root `add-menu` and keep styling minimal inline-free (a few CSS
  rules injected once via a `<style>` tag from addMenu.ts is fine; P1-7 themes it).
- Click an entry → `scene.add(def.name, def.make())`, `selectOnly` it, push
  `new AddObjectsCommand('Add ' + def.name, scene, [obj])`, close the menu,
  set status `Added <name>`.
- `Escape`, clicking elsewhere, or opening again closes it. Menu must not leak
  listeners (remove on close).
- New objects spawn at the origin.

## Duplicate (Shift-D)
- With a non-empty selection: for each selected object create
  `scene.add(dupName, obj.mesh.clone())`, copy `transform` and `visible`.
- Name: Blender-style suffix — `Cube` → `Cube.001`, `Cube.001` → `Cube.002`
  (strip an existing `.NNN`, then lowest unused 3-digit suffix scene-wide).
- Select only the duplicates, push one `AddObjectsCommand('Duplicate', …)`,
  then immediately `startOperator(new TranslateOperator())` so the copies ride
  the pointer (Blender behavior). The move is its own undo step — that's fine.

## Delete (X)
- `X` (no modifiers, no active operator, selection non-empty):
  `undo.push(DeleteObjectsCommand.perform('Delete', scene, [...scene.selection]))`,
  status `Deleted N object(s)`. No confirmation menu.

## Acceptance criteria (verify via headless CDP like e2e/smoke.mjs)
1. Shift-A opens the menu with 6 primitives; clicking "UV Sphere" adds a second object, selected; Ctrl+Z removes it; Ctrl+Shift+Z brings it back selected.
2. Shift-D duplicates the cube (`Cube.001`), starts a Move that LMB confirms elsewhere; Ctrl+Z undoes the move, second Ctrl+Z removes the duplicate.
3. X deletes the selection; Ctrl+Z restores it (same position, still selected).
4. Escape closes the menu without adding; keymap ignores Shift-A while a modal operator runs.
5. `npm run build` and `npm test` pass (existing tests untouched).
