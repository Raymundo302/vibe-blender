## Nodes (deterministic, pure)
- **imageTexture**: input uv(vector), param image('image' kind, default
  null), output color. Bilinear sample (copy the bilinear idiom from
  tracer.ts's sampleImageBilinear — do NOT import tracer code into core;
  reimplement locally, raw values). v-flip: pixels row 0 = top, uv v=1 = top
  (the F13-1 convention — check how the tracer samples texImage and match
  it). No image / not decoded → [1,1,1].
- **colorRamp**: input fac(float 0.5), param ramp('ramp' kind, default
  `{ stops: [{pos: 0, color: [0,0,0]}, {pos: 1, color: [1,1,1]}] }`), output
  color. Linear interpolation between sorted stops; fac clamped to [first.pos,
  last.pos]. Tolerate malformed param (fallback default ramp).
- **math**: inputs a(float 0.5), b(float 0.5), param
  op(select: add/subtract/multiply/divide/power/minimum/maximum, default
  'multiply'), output value(float). divide by |b|<1e-9 → 0; power of
  negative base with fractional exp → 0 (document).

## Result
Built. Three deterministic, pure nodes registered in NEW `src/core/nodes/nodesB.ts`
(one sanctioned import line added to `builtins.ts`):

- **imageTexture** — uv(vector) input, image('image') param, color output. Bilinear
  clamp-to-edge sample reimplemented locally from tracer.ts's `sampleImageBilinear`
  (core never imports render code). v-flip: image pixels row 0 = top but uv v=1 = top
  (Blender / F13-1), so it samples row `1 - v`. Missing / null / undecoded image → `[1,1,1]`.
  The uv socket carries a **NaN sentinel default** so eval distinguishes an unconnected
  socket (→ ctx.u/ctx.v) from a connected upstream value (always real numbers) — the
  evaluator passes socket defaults through `coerce` for vector sockets untouched.
- **colorRamp** — fac(float 0.5) input, ramp('ramp') param, color output. Stops sorted
  internally; linear interp between the bracketing pair; fac clamped to
  [first.pos, last.pos]. Malformed param (missing/empty/bad-typed stops) → default
  black→white ramp.
- **math** — a/b(float 0.5) inputs, op(select: add/subtract/multiply/divide/power/
  minimum/maximum, default multiply) param, value output. `divide` by |b|<1e-9 → 0;
  `power` of a negative base with a fractional (non-integer) exponent → 0 (no real
  result); any non-finite result sanitized to 0; unknown op → multiply.
