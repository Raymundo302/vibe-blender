## Tracer hook

area вЂ” read it first): if `mat.nodeGraph` is set, call evaluateGraph at the
interpolated hit UV with `images: mat.nodeImages ?? undefined` and OVERRIDE
baseColor / metallic / roughness / emissiveГ—emissiveStrength before the BSDF
math. The P13 map-slot and base-texture paths are SKIPPED for node materials
(the graph is the whole truth вЂ” Blender semantics). Bit-stability rule (same

## Node set A (nodesA.ts, all deterministic, no Math.random)
- **checker**: inputs uv(vector, default from ctx? NO вЂ” input socket default
  [0,0,0] means "unconnected"; when unconnected USE ctx.u/ctx.v), params
  scale(float, default 8), colorA/colorB(color params, defaults 0.2-grey /
  1.0-white). Output color. Parity of floor(uВ·scale)+floor(vВ·scale) вЂ” match
  the existing checker in renderedPass/tracer so a graph checker looks like a
  texKind checker at the same scale.
- **noise**: fbm value noise, params scale(default 5) octaves(float 1-8
  default 3, floor it), seeded by a fixed integer hash of the lattice coords
  (mulberry32-style hash вЂ” copy the hashing idiom used by the Scatter
  modifier if convenient). Outputs: value(float 0..1), color(grey triplet).
  Same (u,v,params) в†’ same output, always.
- **mixColor**: inputs a(color), b(color), fac(float 0.5). Output color:
  aВ·(1в€’fac)+bВ·fac, fac clamped 0..1.
For checker + noise: when their uv INPUT is connected use it (take .xy of the
vector), else fall back to ctx.u/ctx.v. Document this convention in the file вЂ”
P14-3's image node follows it.
