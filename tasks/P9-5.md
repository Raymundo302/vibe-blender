## Goal

mesh's faces with seeded randomness — the donut's sprinkles, as a modifier.

## Modifier
registerModifier('scatter', 'Scatter', ...). Params (all through fields()):
- source ('object' kind, default -1) — the object to instance (its EVALUATED
  mesh via ctx.target)
- count (int, default 100, 0..2000)
- seed (int, default 0)
- scale (number, default 1) + randomScale (number 0..1, default 0.2 —
  per-instance scale jitter fraction)
- alignNormal (bool, default true) — orient instance +Y along the surface
  normal (build an orthonormal basis; deterministic tangent choice)
- randomRotation (bool, default true) — random spin around the normal
- offset (number, default 0) — lift along the normal
- minDistance (number, default 0 — 0 disables) — dart-throwing rejection:
  candidates closer than this to an accepted point are skipped (generate
  candidates until count accepted or 10× count attempts; document)
- upOnly (bool, default false) — only faces whose world-ish (host-local)
  normal has y > 0 (the "no sprinkles under the icing" mask)
- colorVariation (number 0..1, default 0) — when > 0, each instance's faces
  get ONE random pastel faceTint (pick hue from seeded RNG, convert HSL(h,
  0.55, 0.72)→RGB, lerp strength by the param). Uses the P9-core faceTints.

apply(): no ctx / unresolved source / source === host → input unchanged
(document; also guard count 0). Otherwise: area-weighted random face pick +
uniform barycentric sample (seeded mulberry32 — NO Math.random), build each
instance by cloning source verts/faces into the OUTPUT mesh (host mesh copied
first, instances appended in deterministic order) with the composed transform
(scale · rotation · translation in host-local space; use inv(hostMatrix) ·
targetMatrix? NO — the source's own transform is IGNORED like Blender's
"reset transform" instancing; instances live in host-local space at the
sampled surface points). Copy the source's creases per instance; skip its
tints when colorVariation > 0 (yours win). depVersion: source target version +
host matrix (offset/normals are host-local so host matrix actually doesn't
matter — include just the target version; justify in Result).

## Acceptance criteria
1. Unit: count N on a plane → base plane + N instances (vert/face math);
   same seed → byte-identical output (serialize both, compare); different
   seed differs; minDistance enforced (no accepted pair closer); upOnly on a
   cube host → instances only on the +Y face; colorVariation > 0 → each
   instance's faces share one tint, instances differ (≥2 distinct tints for
   20 instances); no ctx → identity.
2. e2e: torus + small cylinder via __app, Scatter through the Modifiers tab
   (source dropdown), assert evaluated face count grew by count×source-faces;
   rendered mode screenshot shows multiple distinct sprinkle colors with
   colorVariation 1 (sample a few pixels); save→load round-trip keeps the
   source reference and the identical evaluated result (serialize twice).
3. All e2e suites (flock) + npm test + npx tsc --noEmit pass.

Design notes:
- Determinism: single mulberry32(seed) stream (no Math.random). Placement
  phase draws a FIXED 4 values per candidate attempt (area-weighted face pick,
  in-face fan-triangle pick, two barycentric coords) so the dart-throwing
  rejection loop stays reproducible regardless of accept/reject; per-instance
  appearance (spin, scale jitter, color hue) is drawn in a second pass over the
  accepted points in order. Same seed+params → byte-identical output (verified
  by serializing two runs).
- minDistance: dart-throwing rejection, up to `max(count, count*10)` attempts,
  reject a candidate within minDistance of any already-accepted point; may
  yield fewer than `count` when the surface is crowded (documented in-file).
- Space: instances live in HOST-LOCAL space at the sampled surface points; the
  source's own world transform is IGNORED (Blender "reset transform"
  instancing). Composed vertex = translate(surfacePoint + normal·offset) ·
  rotate(orthonormal basis, +Y→normal when alignNormal, else world +Y, with
  optional spin around that up axis) · scale(scale·(1+jitter)). Source creases
  are copied per instance (remapped to the new vert ids). Source tints are
  copied when colorVariation==0; when colorVariation>0 they're skipped and each
  instance's faces get ONE seeded pastel HSL(h,0.55,0.72)→RGB tint, lerped from
  white by the colorVariation strength ("yours win").
- upOnly masks to host faces whose host-local normal has y>0.
- No-op (returns a clone of the input, geometry unchanged) when: no ctx (bare
  mesh / unit tests), source unresolved / -1 / the host itself (the
  modifierContext visited-guard returns null for a cycle back to host),
  count<=0, or the host has no eligible faces / zero total area.
- depVersion: returns ONLY the source target's evaluated-mesh version. The host
  world matrix is intentionally omitted: offset and normals are all host-local,
  so the composed instance positions don't depend on the host's world matrix —
  it cancels. Params ride the host's modifiersVersion and host geometry rides
  mesh.version, so both already invalidate the evaluatedMesh cache.
