## Euler support (needed for the rotation fields)
Add to `Quat` (XYZ intrinsic order, radians — the standard Blender default):
- `static fromEulerXYZ(x: number, y: number, z: number): Quat`
- `toEulerXYZ(): { x: number; y: number; z: number }` — handle the gimbal
  singularity (|sin(pitch)| ≈ 1) gracefully.
Tests: fromEuler→toEuler roundtrips a grid of angles (avoid the singularity in
the grid, test the singularity separately); fromEulerXYZ(0,0,0) is identity;
rotating Vec3.X by fromEulerXYZ(0,0,π/2) ≈ Vec3.Y.

## Panel behavior
- Shows the **active** object (`scene.activeObject`); when null, a muted
  "No active object" placeholder.
- Fields, three-column rows labeled X/Y/Z:
  - Location (world units, 3 decimals)
  - Rotation (degrees, 1 decimal — converted via the euler helpers)
  - Scale (3 decimals)
  Plus the object name as a read-only header line (rename lives in the outliner).
- `<input type="number">` per component (step 0.1; rotation step 1).
- Commit on `change` (Enter or blur): build the new `Transform` with the
  parsed values, push `new TransformCommand('Set Transform', [{ object, before, after }])`
  (state applied first, then push — the standard convention). Ignore NaN input
  (restore the displayed value).
- Live refresh in `update()`: rewrite input values each frame from the active
  object's transform UNLESS that input (or any input in the panel) is focused;
  also skip rewriting when the value string is unchanged (avoid caret churn).
- Switching active object repopulates immediately.

## Acceptance criteria (headless CDP)
1. Panel shows Cube's transform; moving the cube with G updates Location fields live.
2. Typing `2` into Location X (dispatch input+change events) moves the cube to x=2 in the viewport; Ctrl+Z moves it back AND the field shows the old value.
3. Rotation field: entering 90 in Z rotates the cube (verify `__app.scene.objects[0].transform.rotation` ≈ fromEulerXYZ(0,0,π/2)); the displayed euler values survive a G-move untouched.
4. With nothing selected (`__app.scene.deselectAll()`), the placeholder shows.
5. Quat euler unit tests pass; `npm run build`, `npm test`, `node e2e/smoke.mjs` all pass.
