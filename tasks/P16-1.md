## Behavior
🎞 (and Ctrl+F12 keydown registered by YOUR module on window — check it's
free) opens a small modal: mode (WebM video / PNG sequence), fps (default
scene.fps), Start/End (default scene range), Render + Cancel. Rendering:
loop start→end; per frame applyAnimation → force a Rendered-mode
renderer.render pass → capture the canvas. WebM: canvas.captureStream(0) +
MediaRecorder, requestFrame per rendered frame, timestamps at fps; PNG mode:
toBlob per frame into a minimal STORE-only (no compression) ZIP writer you
implement locally (~100 lines, CRC32 table) → one .zip download. Progress bar
+ frame counter + working Cancel (abort cleanly). Restore frameCurrent,
playing and prior shading mode after. Refuse when start ≥ end (status msg).
