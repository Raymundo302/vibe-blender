## Shared numeric input (Blender behavior)
While modal, typing builds a number that overrides the pointer: digits, `.`,
leading `-` (minus toggles sign), `Backspace` deletes (empty string returns to
pointer control). Implement once in `numericInput.ts`:

```ts
export class NumericInput {
  /** Returns true if the key was consumed. */
  handleKey(key: string): boolean;
  /** Parsed value, or null if empty/not a number. */
  get value(): number | null;
  /** Raw string for the status bar ('' when empty). */
  get text(): string;
}
```

Both operators consume keys in `onKey` in this order: X/Y/Z axis toggle first,
then NumericInput. When `value` is non-null it replaces the pointer-derived
angle/factor and the status bar shows the typed text.

## RotateOperator semantics
- Pivot = median of selected objects' positions (like translate).
- Angle from pointer: project the pivot to CSS px (formula in conventions);
  angle = signed angle swept by the pointer around that screen point since
  `start` (atan2 of current vs start vector; accumulate across the ±π seam so
  dragging keeps going past 180°).
- Axis: default = camera `forward` (view axis). `X`/`Y`/`Z` toggles a world
  axis lock (press again to return to view axis).
- Numeric input = angle in **degrees**.
- Apply to every target, from its `before` state each move (never accumulate):
  `q = Quat.fromAxisAngle(axis, angle)`;
  `position' = pivot + q.rotate(before.position − pivot)`;
  `rotation' = q.mul(before.rotation)`. Scale unchanged.
- Status: `Rotate  -37.50°  [Y axis] ...` matching translate's format, with
  the lock hint and confirm/cancel hint.

## ScaleOperator semantics
- Pivot = median of selected positions.
- Factor from pointer: `dist(pointer, pivotScreen) / dist(startPointer, pivotScreen)`
  (guard the denominator; if the start distance is < 2px use 2px).
- Default: uniform. `X`/`Y`/`Z` toggles single-world-axis scaling.
- Numeric input = the factor itself.
- Apply from `before` each move. Uniform: `position' = pivot + (before.position − pivot) * f`,
  `scale' = before.scale * f`. Axis-locked (e.g. X): only the X component of both
  the pivot offset and `scale` are multiplied by `f`.
- Status: `Scale  2.000  [uniform] ...` / `[X axis]`, same hint style.
