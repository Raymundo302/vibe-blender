### Two REAL gaps found (ranked in UV-RUN.md; reported, NOT fixed — not my files)
- **#1 Modifiers strip UVs.** No modifier copies `mesh.uvs`, so the evaluated
  subsurf/solidify icing has no UVs → renders untextured. The run clears the icing
  stack to render the unwrapped base cap.
- **#2 Z-fighting.** Clearing the stack drops the shrinkwrap+solidify offset, so
  the icing cap collapses onto the donut body (cloned from it) and the untextured
  torus hides the checkered icing. The run HIDES the torus for the checker
  inspection (raster + F12) and restores it for the save. (This cost the bulk of
  debugging; the checker and unwrap were correct all along.)
