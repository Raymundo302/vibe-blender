## The run (screenshots to /tmp/uvrun/NN-*.png)
1. Load the donut via the deep-link loader path (`__app.io.apply` on a fetch
   of research/donut.vibe.json — the file predates UVs and must load clean).
2. Icing: edit mode, Alt+click the inner ring edge loop, Ctrl+E Mark Seam,
   then a second seam along the rim; U → Unwrap. Assert every icing face got
   UVs in [0,1]² and ≥ 2 islands exist.
3. Open the UV Editing workspace (or switch an area to UV Editor): islands
   visible (pixel probe), click-select one, G-nudge it, undo.
4. Material tab: icing texture → Checker; rendered viewport shows the checker
   following the icing's curvature (two probe pixels differ); screenshot.
5. Torus (donut body): U → Smart UV Project without seams — UVs populate.
6. F12 path trace at low samples: the icing's checker is visible in the
   render (variance probe over the icing region vs a no-checker control).
7. Save the scene to research/donut-uv.vibe.json; reload; byte-identical
   re-serialize; UVs + texture survive.
