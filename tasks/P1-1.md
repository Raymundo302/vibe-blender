## What to build
Five factory functions returning `EditableMesh` (see `EditableMesh.fromData` in
`src/core/mesh/EditableMesh.ts`), all centered at the origin, Blender-default
proportions, **Y-up** (our grid is the XZ plane — Blender's Z-up defaults are
rotated accordingly), faces wound CCW seen from outside (backface culling is on):

- `makePlane(size = 2)` — one quad in the XZ plane, normal +Y.
- `makeUvSphere(radius = 1, segments = 32, rings = 16)` — poles on the Y axis; quad rings, triangle fans at the poles.
- `makeCylinder(radius = 1, depth = 2, segments = 32)` — axis along Y; side quads plus two n-gon cap faces (EditableMesh supports polygon faces — use one face per cap).
- `makeTorus(majorRadius = 1, minorRadius = 0.25, majorSegments = 48, minorSegments = 12)` — lies flat in the XZ plane (hole up the Y axis); all quads.
- `makeIcoSphere(radius = 1, subdivisions = 2)` — start from an icosahedron, subdivide each triangle 4-way `subdivisions` times, project verts to the sphere. Share subdivision midpoints (no duplicate verts).

Also export the registry the Add menu (P1-6) will consume:

```ts
export interface PrimitiveDef { name: string; make: () => EditableMesh; }
export const PRIMITIVES: PrimitiveDef[]; // Plane, Cube, UV Sphere, Ico Sphere, Cylinder, Torus
```

## Acceptance criteria (encode as vitest tests in primitives.test.ts)
1. Vert/face counts are exactly right for default params (derive the formulas; e.g. uv-sphere: `segments * (rings - 1) + 2` verts; torus: `majorSegments * minorSegments` verts and faces; ico-sphere: 42 verts / 80 faces at subdivisions = 1).
2. Every mesh is manifold-or-boundary: each edge (`mesh.edges()`) borders exactly 2 faces — except the plane, whose 4 boundary edges border 1.
3. Normals point outward: for sphere/torus/cylinder, `mesh.faceNormal(f)` dotted with (face centroid − nearest-axis-point) > 0 for every face. For the plane, the face normal is +Y.
4. All verts lie at the expected radius (spheres: `|co| ≈ radius` within 1e-6; torus: distance from the ring circle ≈ minorRadius).
5. `PRIMITIVES` has 6 entries and every `make()` returns a fresh, non-empty mesh.
6. `npm run build` and `npm test` pass.
