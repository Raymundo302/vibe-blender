# P3-1 — OBJ export + import

Read `tasks/P3-CONVENTIONS.md` first. Runs after P3-2 (reuse `topbar-btn` and
`e2e/ship.mjs`).

## Goal
Topbar "Export OBJ" downloads all visible objects as a Wavefront .obj
(world-space, one `o` block per object). "Import OBJ" loads a .obj as new scene
object(s) with undo.

## Files
- NEW `src/io/obj.ts` — `exportObj(scene: Scene): string`,
  `parseObj(text: string): { name: string; positions: [number,number,number][]; faces: number[][] }[]`
  (throws readable errors on garbage)
- NEW `src/io/obj.test.ts`
- `src/ui/topbar.ts` — buttons (`data-action`: `export-obj`, `import-obj`)
- `src/main.ts` — wiring; expose on `window.__app.io`
- `e2e/ship.mjs` — append checks

## Export rules
- Header comment `# Vibe Blender`. For each VISIBLE object: `o <name>`, `v` lines
  (transform applied — world space, `toFixed(6)` trimmed of trailing zeros), `f`
  lines with GLOBAL 1-based indices (OBJ indices accumulate across objects),
  polygons as-is (no triangulation). No vt/vn/mtl.

## Import rules
- Parse `v` and `f` only (skip vt/vn/vp/mtl/s/usemtl lines silently); support
  `f a b c`, `f a/b/c` forms (take the vertex index), and NEGATIVE indices
  (relative, per spec). Objects split on `o`/`g` lines; no `o`/`g` → one object
  named from the file ("imported").
- Each parsed object → `scene.add(name, mesh)` with identity transform (verts
  already world-space), then one `AddObjectsCommand('Import OBJ', ...)` for the
  whole import, objects selected, last one active.

## Acceptance criteria
1. Unit: cube exports to exactly 8 `v` + 6 `f` lines with correct indices; a
   2-object scene's second object's `f` indices are offset by the first's vert
   count; parseObj round-trips our own export (counts + positions); negative
   indices and `a/b/c` forms parse; garbage input throws.
2. e2e: `__app.io.exportObj` string has `o Cube`; import a small OBJ string
   (through the same code path the file input uses) → new object appears
   selected, Ctrl+Z removes it, Ctrl+Shift+Z restores it.
3. `npm run build`, `npm test`, all three e2e files pass.
