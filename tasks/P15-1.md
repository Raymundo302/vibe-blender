## The pane (editor type 'timeline', title 'Timeline')
A short horizontal strip (worksplace areas can be resized; render fine at any
height ≥ 90px):
- Header row: ⏮ (to frameStart), ▶/⏸ toggle, frame number input (editable),
  Start/End inputs, and the current fps as static text.
- Ruler + track area (canvas): frame ticks (every 5/10 depending on zoom —
  fixed zoom fit-to-range is fine for v1: map [frameStart, frameEnd] to the
  canvas width), a playhead line at frameCurrent, and ONE row per SELECTED
  object showing its name + a diamond per keyed frame (union of all its
  fcurves' key frames; filled diamond when EVERY LocRotScale channel has a
  key there, hollow otherwise).
- Click/drag on the ruler or track scrubs: sets scene.frameCurrent (rounded,
  clamped to [start, end]) and calls `applyAnimation(scene, frameCurrent)`.
  NOT undoable (Blender semantics).
- Playback: while scene.playing, advance in the pane's update() using
  performance.now() deltas × scene.fps, looping start↔end; each advance
  applies the sampler. ▶/⏸ and Spacebar flip scene.playing.
- Start/End inputs write scene.frameStart/End (clamp: start ≥ 0, end >
  start). Frame input scrubs like a click.
